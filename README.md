## TERMUX USER
```bash
> pkg install bash
> pkg install nodejs
> pkg install libwebp
> pkg install git
> pkg install imagemagick
> pkg install ffmpeg
> termux-setup-storage
> cd /sdcard/LucBot
> git clone https://github.com/siegrin/LucBot.git
> cd LucBot
> npm start
```
